# Photo Portfolio Manager

Repository for the photography portfolio system, featuring a React frontend, a Python Azure Functions backend, and infrastructure as code.

## Repository structure

- `.github/workflows/` - CI/CD pipelines
- `infra/` - Infrastructure as code (Bicep or Terraform)
- `frontend/` - React application using Vite + TypeScript
- `backend/` - Azure Functions in Python

## Next steps

1. Initialize the Git repository
2. Configure the React project in `frontend/`
3. Configure the Azure Functions backend in `backend/`
4. Create the infrastructure scripts in `infra/`
